Information about html .


HTML "Hyper text markup language" is the standard language for creating webpages. It uses tags to structure content like text, image, link, table, and form, HTML is not a programming languages but markup language that defines the layout of a webpage. It works together with Css (stylin) and JavaScript (functionality).

HTML tags

i) "html" tag :- Defines the beginning and end of an HTML document.
ii) "head" tag :- Contains matadata like tile, Css, Scripts,etc,( not visible on page).
iii) "body" tag :- Holds all the visible contant of the webpage.
iv) "heading" tag :- heading, "h1" is the largest, "h6" the smallest.
v) "p" tag :- used to write paragraph of text .
