C# (C-Sharp) :-

• C# is modern, object-oriented programming language developed by Microsoft in 2000.
• It runs on the .NET framework
• It is widely used for buildingDesktop apps, Web apps, Mobile apps, and Games,(With Unity) .
• It combines the power of C++ with the simplicity of Java.

key words oh C#

i) Object - oriented:- Supports concepts like classes, inheritance, polymorphism.
ii) Type - Safe :- prevents type errors at runtime.
iii) Garbege Collection :- Automatic memory manegement.
iv) Cross - Platform :- Runs on windows, Linux, macOS via, .NET Core/.NET 5+.
v) Rich Library Support :- Provide a large set of built-in libraries.
